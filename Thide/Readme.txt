(THIDE) Taskbar Hide -  é um aplicativo leve para Windows 10/11 que permite esconder ou mostrar completamente a barra de tarefas do sistema, incluindo a system tray. Ele funciona perfeitamente com setups customizados usando YASB, deixando o desktop mais limpo, minimalista e organizado.


### Comandos para executar no terminal

# Se instalado via MSI, pode ser executado de qualquer lugar
thide start

# Mostrar a taskbar
thide show

# Esconder a taskbar
thide hide

# Encerrar o aplicativo e restaurar a taskbar
thide stop

# Ativar inicialização automática com o Windows
thide enable-autostart

# Desativar inicialização automática
thide disable-autostart

# Mostrar ajuda
thide help
