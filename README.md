# Dotfiles
Here are the settings for two applications that I use on my Windows machine, which I like because they give it a look close to that of a Linux distribution with a window manager. The programs used were: YASB and Komorebi.

- [YASB](https://github.com/amnweb/yasb?tab=readme-ov-file)
- [Komorebi](https://lgug2z.github.io/komorebi/index.html)
